Tutorial Memasukkan SFX dan VFX

Cara Memasukkan Sound FX

  1. Buka folder `/sfx` di direktori project Anda
  2. Pastikan file suara memiliki format yang didukung (mp3, wav, dll)
  3. Salin file suara ke dalam folder `/sfx`
  4. Nama file sebaiknya deskriptif, contoh: `explosion_loud.mp3`

Cara Memasukkan Video FX

  1. Buka folder `/vfx` di direktori project Anda
  2. Pastikan file video memiliki format yang didukung (mp4, webm, dll)
  3. Salin file video ke dalam folder `/vfx`
  4. Nama file sebaiknya deskriptif, contoh: `smoke_transition.mp4`
